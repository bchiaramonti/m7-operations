# m7-plano-acao

Registro e acompanhamento de Acoes de Melhoria da M7 via conversacao. Opera diretamente no CSV que alimenta o dashboard de gestao.

## Skills

| Skill | Descricao | Invocacao |
|-------|-----------|-----------|
| `managing-plans` | CRUD: criar, atualizar, concluir, cancelar, followup | Automatica via conversacao ou `/managing-plans` |
| `reviewing-plans` | Briefing diario e KPIs do periodo | Automatica via conversacao ou `/reviewing-plans` |

## Dependencias externas

- `plano-de-acao.csv` — Acoes de melhoria (24 colunas)

Caminho base: `OneDrive/desempenho/02-Controle/Planos-de-Acao/`
